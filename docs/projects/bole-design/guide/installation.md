# 安装

| Tables        |      Are      |  Cool |
| ------------- | :-----------: | ----: |
| col 3 is      | right-aligned | $1600 |
| col 2 is      |   centered    |   $12 |
| zebra stripes |   are neat    |    $1 |

推荐使用 pnpm 或 yarn 的方式进行安装：

```shell
# 选择一个你喜欢的包管理器

# NPM
$ npm install bole-design --save

# Yarn
$ yarn add bole-design

# pnpm
$ pnpm install bole-design
```
