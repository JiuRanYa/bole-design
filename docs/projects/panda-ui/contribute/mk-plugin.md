## Markdown Plugin

在您书写Markdown文档时，Panda UI内置了一些markdown语法来帮助您轻松完成提示等功能

### 使用Tip

```markdown
:::tip

使用tip效果

:::
```

:::tip

使用tip效果

:::

### 使用info

```markdown
:::info

使用info

:::
```

:::info

使用info

:::

### 使用warning

```markdown
:::warning

使用warning

:::
```

:::warning

使用warning

:::

### 展示Demo

在一些情况下，我们需要展示代码的Demo，此时可以使用`demo`的插件完成

在`web/docs/demos`下新建您的`vue`文件并引入，`avatar/basic`为`demos/panda-ui`下的路径

```markdown
:::demo

avatar/basic

:::
```
